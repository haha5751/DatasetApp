Welcome.


This app allows users to visualize the air quality data for a specific country.


This app was built using the Django framework.


It features Django templates on the front end using only HTML, CSS and Pythonic logic.


The app should be connected to a PostgreSQL data base. It is queried using the Django ORM.


The data set contains air quality data from various countries around the world. Not all countries are featured within the data set.


Please refer to the readme file to learn more about this data.
